# Installation

Install the dependencies with [Bundler](http://bundler.io/):

```bash
bundle install
```

## Build process

Install [UIkit](https://getuikit.com/) font end framework dependency via Npm:

```bash
npm install
```

Enable live browser reload with the following:

```bash
bundle exec jekyll s
```

Compile and minify:

```bash
npm run build
```
